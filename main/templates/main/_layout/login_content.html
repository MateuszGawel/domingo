Welcome to Domingo - Service Desk Helper

Domingo is a project created by Service Desk for Service Desk. Currently our job, especially monitoring, needs a lot of additional work in external applications. This project will make Service Desk work easier because only Domingo, Jira and E-mails will be necessary.

The project is built from some smaller modules where each can be developed separately. Here's a short list of modules which are, or will be created:

Reports
  • Report creation
    You can create monitoring reports like you used to do in excel. There are many additional functions like fetching created tickets from Jira or better incident management. Just check it out yourself.
  • Report browsing
    You can browse yours and others reports, incidents, alerts, communications or actions. It can help when you'll have problem with some tickets. Thanks to ticket connections and other comments it will be much easier to find out a solution.

Announcements
  • This module will allow to create and read important announcements from other users. You will never miss important news during your night shifts again :)

Mailing
  • Sending all those deploy reminders, incident reports etc. is really uncomfortable task. When this module is finished you will be able to send any information to clients using only few clicks. You will be sure that you have the most recent templates and you will be able to create and edit them with ease.

Monitoring Schedules
  • Excel on SVN won't be needed anymore. Using Domingo you will be able to plan, request changes in schedules to other users and manager will be able to see every shift exchange. It will also have additional funcionality to plan holidays.

Deploy Schedules
  • Also google calendar won't be needed anymore. You will be able to see everything in one place and get reminders during your monitoring shift.

And others
  • We are opened for new ideas and feedback for existing ones. With your help we can make Domingo to be great and making our job much easier project.
